    About

    Description

    This is just a fun project using Rust + WebAssembly. You can generate a Markov Chain using your own text files or some predefined files. This Markov Chain can then be used to generate new texts with sometimes funny results.

    The code

    The source code can be found on GitHub. Feel free to use it for your own research. It is not very optimized and not idiomatic Rust but it works quite well.

    Privacy

    No data is shared with the server. All calculation is done on your local machine. The results are not stored anywhere.
